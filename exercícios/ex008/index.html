Principais formatações

Negrito

Nessa frase, temos um trecho em negrito usando a tag B (não semântica)

Nessa frase, temos um termo em destaque usando a tag STRONG (semântica/tem sentido)

Itálico

Nessa frase, temos um trecho em itálico usando a tag I (não semântica)

Nessa frase, temos um termo em ênfase usando a tag EM (semântica)

Texto marcado

Podemos também criar um texto marcado usando a tag MARK

E no outro parágrafo, temos outro texto marcado no final

Texto grande e texto pequeno

Nesse parágrafo, estamos criando um texto grande com a tag BIG e um texto pequeno com a tag SMALL

Texto deletado

Podemos marcar um texto como excluído com a tag DEL para indicar que ele deve ser lido, mas não considerado

Texto inserido

Podemos marcar um texto como inserido com a tag INS para dar ênfase e indicar que ele foi adicionado depois

Existe também o texto sublinhado que usa a tag U (não semântica)

Texto sobrescrito

Para inserir coisas do tipo x20+3, usamos a tag SUP

Texto subescrito

Para inserir coisas do tipo H20, usamos a tag SUB
